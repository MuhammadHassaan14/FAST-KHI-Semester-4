File and Directory Commands

Command Description
pwd Show current directory
ls List files and directories
cd <directory> Change directory
mkdir <directory_name> Create a new directory
rmdir <directory_name> Remove a directory
touch <filename> Create a file
cat <filename> View file contents
cp <source> <destination> Copy a file
mv <source> <destination> Move or rename a file
rm <filename> Delete a file

Memory Usage Commands

Command Description
free -m Check memory usage in MB
free -g Check memory usage in GB
cat /proc/meminfo Check detailed memory usage
ps aux --sort=-%mem | head Show memory usage per process
vmstat 1 Monitor memory usage live

CPU Usage Commands

Command Description
top Check CPU usage
htop Interactive CPU and process monitor
mpstat CPU usage summary
lscpu Check CPU details
sar -u 1 5 Monitor CPU usage over time

Process Management Commands

Command Description
ps aux List running processes
pstree List all processes in tree format
kill <PID> Kill a process by PID
kill -9 <PID> Force kill a process
pkill <process_name> Kill a process by name
pgrep <process_name> Check a specific process by name
top Monitor real-time processes
ps fax Show process hierarchy
cat /proc/<PID>/status Check process details by PID
netstat -tulnp | grep <port>
or
ss -tulnp | grep <port>		 Find which process is using a port

User and System Information

Command Description
su <username> Switch user
sudo <command> Run command as root
whoami Show current user
who Show active users
uptime Show system uptime

Disk and Network Commands

Command Description
df -h Show disk usage
free -m Show memory usage
find /path -name <filename> Find a file
grep 'text' <filename> Search text in a file
ping <website> Check network connectivity
ifconfig
or
ip a		 Show IP address (ifconfig is deprecated)
wget <URL> Download files

File Compression Commands

Command Description
tar -xvf <file.tar> Extract tar files
tar -czvf <file.tar.gz> <directory> Compress files to tar.gz

Help Commands

Command Description
man <command> Show manual of a command
history Show command history

Sorting and Piping Commands

Command Description
sort <filename> Sort a file alphabetically
sort -r <filename> Sort a file in reverse order
sort -n <filename> Sort by numeric values
sort -u <filename> Sort and remove duplicates
command1 | command2 Use pipe to filter output
command > output.txt Send output to a file
command >> output.txt Append output to a file
command | grep 'text' Use pipe to search text

Date and Time Commands

Command Description
date Display current date and time
date '+%Y-%m-%d %H:%M:%S' Format date output
date --date='yesterday' Show yesterday’s date
date --date='tomorrow' Show tomorrow’s date
date -d @<timestamp> Convert timestamp to date
